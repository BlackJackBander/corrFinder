# corrFinder

**corrFinder** — это аналитический инструмент для поиска корреляции и ковариации между различными акциями на фондовом рынке.

### Основные возможности:
*   **Анализ:** Расчет статистических показателей для выявления взаимосвязей между активами.
*   **Интерфейс:** Современный UI, построенный на компонентах Radix UI и Material UI.
*   **Разработка:** Использование TypeScript обеспечивает надежность кода и типизацию данных.

### Технологии:
*   **Стек:** React, TypeScript, Vite.
*   **Стили:** Tailwind CSS, PostCSS.
*   **UI компоненты:** Radix UI, MUI.
